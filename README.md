# 🚀 Installation

## Install using HACS

1. Open **HACS**

2. Open **Custom repositories**

Add:

```
https://github.com/PakyITA/phoenix-ai-trader
```

Category:

```
Integration
```

Install **Phoenix AI Trader**

Restart Home Assistant.

---

# ⚙️ Configuration

During setup you'll be asked to configure:

- Status file path
- Update interval
- Currency
- Telegram notifications (optional)

Default status file:

```
/config/www/phoenix/status.json
```
